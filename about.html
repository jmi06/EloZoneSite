What is EloGrid?

EloGrid is an open-source data project that provides chess-like NFL team elo ratings. A teams elo rating is generated after each game, their change in elo is based on their opponents rating, the result of the game, and the score differential.

Open Source

The code used for fetching the data, rating calculation, and our X bot can be found on GitHub

Live Updates

Game results accompanied by how they affect a teams rating can be found on our X page. Live results operate on about a 20 minute delay.

The Formula

The Elo formula takes into account the point differential of a game, big win = big gain. This equation was generated by ChatGPT 4o.

Each team starts the season with a rating of 1000

team_1_winprob = 1 1 + 10 ( team_2_elo_before - team_1_elo_before ) 400
team_2_winprob = 1 1 + 10 ( team_1_elo_before - team_2_elo_before ) 400
K = 16 √ó ( 1 + 0.1 √ó points_diff game )
team_1_newrating = team_1_elo_before + K √ó ( team1W - team_1_winprob )
team_2_newrating = team_2_elo_before + K √ó ( team2W - team_2_winprob )
